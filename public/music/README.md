# Background Music

Add a background music file here to make the experience more special!

## How to Add Background Music:

1. Choose a romantic or meaningful song (MP3 format recommended)
2. Name it: `background-music.mp3`
3. Place it in this folder (`public/music/`)
4. The music toggle button will automatically play it

## Music Guidelines:

- **Format:** MP3, WAV, or OGG
- **Length:** 3-5 minutes (it will loop automatically)
- **Volume:** The app sets it to 30% by default
- **File size:** Keep it under 5MB for faster loading

## Where to Find Music:

- **Free Music:** [YouTube Audio Library](https://www.youtube.com/audiolibrary)
- **Royalty-Free:** [Incompetech](https://incompetech.com/music/)
